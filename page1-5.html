사용 제한

NAVER 지도 API v3은 약관에 따라 무료로 제공되고 있습니다. 그러나 다음과 같은 쿼리 할당량 제한이 있습니다.

  • 200,000/일

쿼리 할당량 증대하기

기본적으로 제공하는 쿼리 할당량보다 더 많은 쿼리 할당량이 필요한 경우에는 제휴 신청을 해야 합니다. 제휴 신청은 NAVER 지도 API v3에 대해 일 허용 쿼리 할당량을 상향하는 절차입니다.

제휴 신청이 필요한 경우

  • 애플리케이션에 적용한 NAVER 지도 API v3의 일 기준 쿼리량이 한도를 초과하거나 기존의 서비스 이용량으로 충분히 추정 가능할 경우

제휴 신청이 필요없는 경우

  • NAVER 지도 API v3을 테스트해 보려는 경우
  • NAVER 지도 API v3의 일 쿼리 사용량이 일 허용 쿼리량 한도 내인 경우

주의사항

  • 제휴 신청 전에 애플리케이션을 먼저 등록해주세요.
  • NAVER 지도 API v3의 용도가 불명확하면 반려됩니다.
  • NAVER 지도 API v3의 활용 화면은 실제 애플리케이션 화면을 캡쳐해 주셔야 원활한 검토가 가능합니다.
  • 제휴 신청은 통상 1~2주 정도 소요됩니다.

NAVER 지도 API v3 제휴 신청 시 유의사항

  • NAVER 지도 API v3 사용 쿼리량 한도 내이지만 상업적 목적일 경우는 제휴 신청 후 승인을 받으셔야 합니다.
  • 상업적 목적은 회사 내부 업무용이나 애플리케이션을 통해 사용자에게 비용을 받거나 광고 수익이 발생하는 경우입니다.
